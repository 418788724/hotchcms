# Hotchcms

使用koa, mongodb 开发一套内容管理系统，旨在学习。

## 环境

* **node ^7.0.0**
* npm ^4.0.0
* mongodb ^3.0.0

## Run

1. `$ git clone https://github.com/luckcoding/hotchcms.git`
2. `$ npm install`
3. `$ npm start` or `npm run dev`
